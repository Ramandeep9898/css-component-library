    headings

    i'm also available with .h1 through .h6 classes, for when you want to match the font styling of a heading but cannot use the associated HTML element.

    i'm h1 heading

    i'm h2 heading

    i'm h3 heading

    i'm h4 heading

    i'm h5 heading

    i'm h6 heading

                        <p class="h1">i'm h1 heading</p>
<p class="h2">i'm h2 heading</p>
<p class="h3">i'm h3 heading</p>
<p class="h4">i'm h4 heading</p>
<p class="h5">i'm h5 heading</p>
<p class="h6">i'm h6 heading</p>

    headings with colored text

    Use respective blue-color , green-color, danger-color, info-color, gray-color utility classes to make any text of respective color

    i'm h3 heading with blue text

    i'm h3 heading with green text

    i'm h3 heading with red text

    i'm h3 heading with info text

    i'm h3 heading with gray text 

                    <p class="h3">i'm h3 heading
<div class="gray-color">with gray text</div>
</p>

    text align utilities

    Easily realign text to components with text alignment classes.

    i'm Left aligned text.

    i'm Center aligned text.

    i'm Right aligned text.

                        <p class="text-left">i'm Left aligned text on all viewport sizes.</p>
<p class="text-center">i'm Center aligned text on all viewport sizes.</p>
<p class="text-right">i'm Right aligned text on all viewport sizes.</p>
