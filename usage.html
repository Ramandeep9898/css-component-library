    Get started with Flash.UI

    how to configure Flash.UI in your project

    I'm a HTML-CSS based UI toolkit.
    No need to start from scratch, I'm a open-source component library that help you ship new features faster. Instructions to import me in your HTML or CSS respectively:

    HTML

    <link href="https://flashui.netlify.app/main.css" rel="stylesheet" />

    CSS

    @import url("https://flashui.netlify.app/main.css");
